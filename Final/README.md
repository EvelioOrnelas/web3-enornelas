## App details

```javascript
Functionality of this website is to login with authenticated user, then be able to see users in database that are created.
Ability to create new users with username and password.
Ability to edit username and or password and also delete users.
```

## Quick Start

```javascript
// clone repo
$git clone git@github.com:EvelioOrnelas/web3-enornelas.git

// cd into app
$cd Final

// Install dependencies for server & client
$npm install && npm run frontend-install

// Run client & server with concurrently
$npm run dev

// Server runs on http://localhost:5000 and client on http://localhost:3000
```
